Defining Key Concepts (Definition List)

Data Structures
A way of storing and organizing data in a computer to enable efficient access and modification.
OOP
A programming paradigm based on the concept of "objects", which can contain data and code.
DBMS
A software system that uses a standard method to store and organize data.
Web Technologies
Various tools and languages used in the development of web applications.
Machine Learning
A subset of artificial intelligence that involves the study of algorithms that can improve automatically through experience.
